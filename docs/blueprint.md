# **App Name**: CineScope

## Core Features:

- Homepage Content Fetch: Fetches and displays movies and TV shows in horizontal sections based on popularity, rating, and release date using the TMDB API. Sections include: Now Playing, Upcoming, Popular, Top Rated movies, Airing Today, On The Air, Popular, Top Rated TV shows and Trending Movies/TV/People.
- Detail Page Generation: Generates dedicated detail pages for movies, TV shows, and people with complete information including title, tagline, overview, cast, crew, trailers, similar content, and watch providers.
- Genre-Based Discovery: Allows users to browse content based on genres fetched from the TMDB API, sorting content by popularity, top rated, and release date.
- Provider-Based Filtering: Enables filtering of content based on available streaming providers.
- Global Search: Allows users to search for movies, TV shows, and people using a global search bar that provides combined results linking to detail pages.
- Trending Content Carousel: Display a horizontally scrolling showcase of popular trending content at the top of the homepage. LLM analyzes recent views and ratings to suggest which entries to place in the carousel, as a tool to suggest appealing, timely content to users.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a rich, cinematic feel.
- Background color: Dark gray (#222222) for a modern dark theme.
- Accent color: Electric purple (#BF00FF) to highlight key elements and create visual interest.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look. 'Inter' is suitable for both headlines and body text.
- Use flat, minimalist icons in a light gray or white to maintain a clean aesthetic. Focus on recognizable symbols for categories and actions.
- Employ a card-based layout with carousels for content display. Ensure responsiveness across devices (mobile, tablet, desktop) using a grid system.
- Implement subtle transitions and animations to enhance user experience, such as hover effects on cards, smooth scrolling, and loading indicators.
